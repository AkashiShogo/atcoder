# AtCoder 練習部屋

![Last Commit](https://img.shields.io/github/last-commit/AkashiShogo/atcoder)
![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?logo=kotlin&logoColor=white)

AtCoder の問題を解くための作業用リポジトリです。

言語ごとにディレクトリを分けて管理しており、**複数言語での実装を通じて言語特性・アルゴリズムへの理解を深める**ことを目的としています。

## 実行方法

`kotlin/src/input.txt` にサンプルケースを貼り付けて、以下のコマンドで実行します。

```bash
cd kotlin
./gradlew a   # A問題を実行
./gradlew b   # B問題を実行
./gradlew c   # C問題を実行
```

## AI の活用について

このリポジトリでは AI（Claude）をコミットメッセージの作成・解法の解説・コードレビューなどのサポートに活用しています。
問題を解く作業（コード実装・提出）は自分自身で行っており、AtCoder のルールには抵触していません。
